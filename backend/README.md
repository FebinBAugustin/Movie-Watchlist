# Movie Watchlist Backend

This folder contains a simple Express + MongoDB backend for the movie watchlist app.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm run dev
   ```

## API Routes

- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/profile
- GET /api/movies
- POST /api/movies
- DELETE /api/movies/:id
